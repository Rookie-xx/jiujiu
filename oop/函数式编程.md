## 函数式编程

```python
"""
    函数式编程思想
"""

class SkillData:
    def __init__(self,id,name,atk_ratio,duration):
        self.id = id
        self.name = name
        self.atk_ratio = atk_ratio
        self.duration = duration

    def __str__(self):
        return "技能:%d,%s,%d,%d"%(self.id,self.name,self.atk_ratio,self.duration)

list_skill = [
    SkillData(101,'乾坤大挪移',5,10),
    SkillData(102,'降龙十八掌',8,5),
    SkillData(103,'葵花宝典',10,2)
]

# 需求1：获取攻击比例大于6的所有技能
def find01():
    for item in list_skill:
        if item.atk_ratio > 6:
            yield item

# 需求2：获取持续时间在4--11之间所有技能
def find02():
    for item in list_skill:
        if 4<item.duration<11:
            yield item            

# 需求3：获取技能名称大于4个字并且持续时间小于6的所有技能  
def find04():
    for item in list_skill:
        if len(item.name) > 4 and item.duration < 6:
            yield item

#将每个变化的条件，单独定义在函数中
def condition1(item):
    return item.atk_ratio > 6

def condition2(item):
    return 4<item.duration<11 

def condition1(item):
    return len(item.name) > 4 and item.duration < 6

# "继承"（隔离变化）
def find(func_condition):
    for item in list_skill:
        #"多态"：调用父(变量),执行子(具体函数)
        #        不同子类重写父类方法，执行逻辑不同
        if func_condition(item):
            yield item  

for item in find(condition2):
    print(item)
```

### lambda 匿名函数

+ 语法：lambda 参数列表:函数体
+ 注意:
  + 参数没有可以不填
  + 方法提只能有一条语句，且不支持赋值语句

```

        注意：函数体自带return
"""
```
