## 重写

+ 内置可重写函数

  > Python 中，以双下划线开头、双下划线结尾的是系统定义的成员。我们可以在自定义类中进行重写，从而改变其行为。
  >
  > \__str__函数：将对象转换为字符串(对人友好的)
  >
  > \__repr__函数：将对象转换为字符串(解释器可识别的)

+ *eval*

  ```python
  """
      内置可重写函数  
  """
  
  class StudentModel:
      def __init__(self,name='',age=0,score=0,id=0):
          self.name = name
          self.age = age
          self.score = score
          self.id = id
  
      # 对象 --> 字符串（随意格式）
      # def __str__(self):
      #     return 'hehe'   
  
      # 对象 --> 字符串（解释器可识别，有格式）
      def __repr__(self):
          return 'StudentModel("%s",%d,%d,%d)'%(self.name,self.age,self.score,self.id)
  
  
  
  s01 = StudentModel('zrq',17,100,101) 
  print(s01)      
  print(str(s01))
  
  str02 = repr(s01)      
  print(str02)
  # 根据字符串执行python代码
  re = eval("1+2*5")
  #exec
  print(re)
  
  #克隆对象
  # reper 返回python格式的字符串
  # eval 根据字符串执行代码
  s02 = eval(repr(s01))
  print(s02.name)
  ```
